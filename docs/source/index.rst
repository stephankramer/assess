Assess
======
Analytical Solutions for the Stokes Equations in Spherical Shells
------------------------------------------------------------------

Assess is a python package that implements a number of analytical solutions to
the Stokes equations in cylindrical and spherical shell domains.

Download and installation
-------------------------
Assess is available from the `Python Package Index (PyPi)
<https://pypi.org/project/assess/>`_, so that it can be installed with a simple

.. code::

    $ pip install assess

It can also be downloaded from `github <https://github.com/stephankramer/assess>`_.
